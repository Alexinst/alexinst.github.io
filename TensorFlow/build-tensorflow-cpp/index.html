5 疑难杂症

  1. bazel 版本问题

    解决办法：bazel 0.19.1 -> 0.21.0

  2. eigen 版本问题

    解决办法：更新 eigen 至 3.3，且添加到 CMakeLists.txt 的搜索目录

  3. 找不到 abseil-cpp

    解决办法（来源）：下载 abseil-cpp 并添加到 CMakeLists.txt 的搜索目录

  4. CMakeLists.txt

    解决办法：include_directories(${TENSORFLOW_DIR}/bazel-genfiles)
